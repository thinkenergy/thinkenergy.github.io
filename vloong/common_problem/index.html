常见问题

登录​

问题：网址无法打开或点击登录后报错，报错信息如下​

解决方案：​

查看浏览器是否默认选用了 https 协议，需要修改为 http 协议，或「点此解决。」

使用​

问题：开发时想使用镜像中没有的依赖包怎么办​

解决方案：​

  • 如果是在 Notebook 中临时使用，Notebook 提供了一种方法可以让用户安装自己所需要的软件包：
    • 打开终端
    • 启动bash
    • 启动相应虚拟环境，这里选择 Python3 的环境
    • 安装所需软件包，这里以安装 h5py为例
    • 安装完成后即可使用对应包进行开发
  • 如果需要创建训练任务，可以通过邮件或微信群「联系我们」，我们会为您更新镜像。

问题：创建 Notebook 时未正常打开JupyterLab，显示503怎么办​

解决方案：​

出现 503 一般是由于初次创建 Notebook，JupyterLab 未及时启动，直接在本页面刷新即可。
