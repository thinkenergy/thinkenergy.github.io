Description of Dataâ€‹

The car charge data is in pkl format, and each pkl file includes two parts: pkl[0], pkl[1] Where pkl[0] is the data part, and it consists of: (8 dimensions in total) ['volt', 'current', 'soc', 'max_single_volt', 'min_single_volt', 'max_temp', 'min_temp', 'timestamp'] . Each field name contains the data of the corresponding field of the battery And where pkl[1] is the label part in OrderedDict format, and it consists of : (4 dimensions in total) [('label', '00'), ('car', 31), ('charge_segment', '177'), ('mileage', 10900.86015625)] Each field name represents the corresponding label of this segment of data.
